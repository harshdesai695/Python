It’s a simple Gui application created using tkinter to generate Qrcode of any link you want.

Module Required:

•	Tkinter  (use command in terminal for installation-> pip install tkinter)
•	os
•	pypng    (use command in terminal for installation-> pip install pypng)
•	pyqrcode (use command in terminal for installation-> pip install pyqrcode)
•	pillow   (use command in terminal for installation-> pip install Pillow)

After Writing a link address in textbox click on generate then a qrcode will be generated and a png will also 
be created and saved in a same folder so that you can share it anywhere
